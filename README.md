# RACV Policy Documents RAG (Retrieval-Augmented Generation)

An intelligent Retrieval-Augmented Generation (RAG) system built using **LangChain** to analyze, query, and retrieve information from **RACV (Royal Automobile Club of Victoria)** policy documents. 

This repository currently serves as an implementation foundation, laying down the document processing pipeline, embedding strategies, and vector storage mechanisms that will power the complete RAG assistant.
